# Kids English

A fun, interactive website to help kids learn basic English vocabulary through 5 colorful lessons.

## Lessons Include:
- 🐶 Animals
- 🎨 Colors
- 👨‍👩‍👧‍👦 Family
- 🔢 Numbers
- 🍎 Food

## How to Use
1. Open the website
2. Click on any lesson
3. Learn vocabulary with emojis and example sentences
